# Agent Instructions

This is the official working folder for the North Slope Gas Hydrates project.

Before making project changes:

1. Read `PROJECT_CONTEXT.md`.
2. Read `docs/PROJECT_ARCHITECTURE_AND_ACTIVITY_MAP.md`.
3. Read the relevant technical documentation named there.
4. Check `git status --short` and preserve user changes.
5. Respect the public-source versus authorized-runtime data boundary.

After meaningful work:

1. Update `docs/PROJECT_ARCHITECTURE_AND_ACTIVITY_MAP.md` when a milestone,
   blocker, dependency, priority, or next action changes.
2. Update `PROJECT_CONTEXT.md` only when the concise project orientation changes.
3. Record only important activity, not every edit.
4. Update any affected technical documentation and tests.
5. Never claim that a file or result exists without verifying it locally.

Do not commit or publish classified, controlled, restricted, credential-bearing,
approved-environment-only, or personally identifying data. Approved well logs,
core data, named restricted identifiers, derived sensitive outputs, populated
runtime configurations, and trained models belong only in the ignored authorized
runtime folders described in `docs/opensciencelab_runtime_layout.md`.
