# Well-Log Spreadsheet Reference Intake

Use this folder for public or synthetic screenshots and workbook references that
define the desired well-log scaffold.

Do not place restricted or approved-environment-only data here.

For each item, add a short note containing:

- source filename and date
- workbook sheet or screen name
- variables and units shown
- formulas or derived fields shown
- chart tracks and visual conventions
- desired website/runtime behavior
- data classification: public, synthetic, or restricted
- questions that still need an answer
