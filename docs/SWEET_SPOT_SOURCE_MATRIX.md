# Sweet-Spot Source Matrix

Last updated: 2026-06-08

## Purpose

Track the evidence used by the synthetic North Slope sweet-spot scaffold without
counting duplicate formats or project synthesis documents as independent
scientific confirmation.

The repository index currently contains 28 candidate artifacts in six source
groups. Four connected Drive documents were reviewed during one synthesis pass;
they are a subset of the working evidence base, not the total source count.

## Evidence Tiers

| Tier | Meaning | Website use |
|---|---|---|
| Primary public science | Peer-reviewed papers, USGS reports, and Alaska public geology publications | Supports scientific relationships and regional constraints |
| Comparative public science | Field studies outside the North Slope that support methods or failure-mode comparisons | Supports methods only; never substitutes for North Slope calibration |
| Project synthesis | Manuscripts, equation maps, range guides, and integrated working documents | Organizes logic and implementation requirements |
| Data and map support | Public GIS, directional surveys, inventories, and recovery notes | Provides context and provenance, not interval-scale hydrate proof |

## Implementation Rule

Every scientific relationship shown publicly should be traceable to at least one
verified primary source. Working thresholds, ranges, and scores must remain
marked synthetic until calibrated against the recovered workbook and authorized
well-log and core data.
